## 1. Line Identity

**Form 1040 line 19** is the **nonrefundable** child tax credit plus credit for other dependents.

```text
Form1040.line19 = Schedule8812.line14
```

Schedule 8812 also computes the **refundable Additional Child Tax Credit (ACTC)** — both produced together:

```text
Form1040.line28 = Schedule8812.line27
```

## 3. 2025 Eligibility Rules

### 3a. Return-level SSN / ITIN rule for claiming CTC or ACTC (G7 + G9 fixes)

Beginning in tax year 2025, the filer identification rule changed:

- Single / HOH / MFS / QSS / NR filer: the taxpayer must have a **valid SSN** to claim CTC or ACTC.
- Joint return: **only one spouse must have a valid SSN**. The other spouse must have either an SSN or an ITIN, issued on or before the due date of the return.
- If this rule fails, the return cannot claim CTC or ACTC.
- This rule does **not** block ODC; ODC uses the normal timely-issued SSN/ITIN requirement.

A valid SSN is one valid for employment and issued before the due date of the 2025 return (including extensions). Backend ITIN check uses 9-leading-digit pattern after digit-strip. **G9 fix**: on MFJ, the block fires only when **both** spouses have ITINs (one valid SSN is sufficient).

### 3b. Child-level CTC rule

A dependent can be used for CTC/ACTC only if **all** of the following are true:

1. The child is claimed as a dependent on the return.
2. The child is **under age 17 at the end of 2025**.
3. The child is a U.S. citizen, U.S. national, or U.S. resident alien.
4. The child has a **valid SSN** issued by the due date of the return.
5. The normal qualifying-child dependency rules are met.
6. The `Child tax credit` box is checked for that dependent on Form 1040.

If the child was born and died in 2025 without an SSN, the IRS allows the credit if the required birth/death/hospital records are attached.

### 3c. ODC rule

A dependent qualifies for ODC if:

1. The person is claimed as a dependent.
2. The person cannot be used for CTC/ACTC.
3. The person is a U.S. citizen, U.S. national, or U.S. resident alien.
4. The person has an SSN, ITIN, or ATIN issued on or before the due date of the return.
5. The `Credit for other dependents` box is checked for that dependent on Form 1040.

A person cannot be counted for both CTC and ODC (structurally enforced by mutually-exclusive if-else in the dependent counting loop).

### 4b. Threshold on line 9

```text
if filingStatus == MFJ:
    line9 = 400000
else:
    line9 = 200000
```

`QSS` does **not** use the joint-return threshold. The 2025 Schedule 8812 threshold is `$400,000` only for MFJ.

### 4c. Counting dependents and tentative credit

```text
line4 = count_of_dependents_checked_for_child_tax_credit
line5 = line4 * 2200                                          (★ 2025 OBBBA)
line6 = count_of_dependents_checked_for_credit_for_other_dependents
line7 = line6 * 500
line8 = line5 + line7
```

Backend iterates the dependents list and calls `qualifiesForCTC()` / `qualifiesForODC()` predicates (delegated to intake form flags for 2025 SSN/ITIN rules).

### 4d. Phase-out reduction

```text
line10 = max(0, line3 - line9)                                (G5 fix — raw excess stored)
line11 = ceil(line10 / 1000) * 50                             ($50 per $1,000 or part of $1,000)
line12 = max(0, line8 - line11)
```

### 4e. Credit limit and line 14

The nonrefundable credit allowed on line 19 cannot exceed the remaining tax allowed by Credit Limit Worksheet A:

```text
line13 = CreditLimitWorksheetA.result
line14 = min(line12, line13)
Form1040.line19 = line14
```

If `line18` of Form 1040 is zero, line 14 will be zero — but this does **not** automatically eliminate ACTC.

## 5. 2025 Credit Limit Worksheet A and Worksheet B

### 5a. Worksheet A

For 2025, Worksheet A starts from Form 1040 line 18 (`totalTaxBeforeCredits`) and subtracts these earlier credits if applicable:

- Schedule 3 line 1 (FTC)
- Schedule 3 line 2 (childcare)
- Schedule 3 line 3 (education)
- Schedule 3 line 4 (saver's)
- Schedule 3 line 5b (energy efficient home improvement)
- Schedule 3 line 6d (elderly/disabled)
- Schedule 3 line 6f (clean vehicle)
- Schedule 3 line 6 (general business)
- Schedule 3 line 6m (previously owned clean vehicle)

Then:

```text
worksheetA.line1 = form1040.line18                            (post-FTC refresh per G-new-1 fix)
worksheetA.line2 = sum_of_the_2025_Schedule3_lines_listed_above
worksheetA.line3 = max(0, worksheetA.line1 - worksheetA.line2)
worksheetA.line4 = worksheetB_adjustment_or_zero
worksheetA.line5 = max(0, worksheetA.line3 - worksheetA.line4)
Schedule8812.line13 = worksheetA.line5
```

**⚠️ G6 open**: CLW-A currently subtracts FTC + childcare + education + saver's + elderly/disabled (5 of 8). Missing: line 5b + line 6f + line 6 + line 6m. Awaits Form 8396 / Form 5695 Part I / Form 8859 implementation.

### 5b. When Worksheet B is required

In 2025, Worksheet B is used only if **all** of the following are true:

1. The return claims one or more of these credits:
   - Form 8396 mortgage interest credit
   - Form 8839 adoption credit
   - Form 5695 Part I residential clean energy credit
   - Form 8859 District of Columbia first-time homebuyer credit
2. The return is **not** filing Form 2555.
3. Schedule 8812 line 4 is greater than zero.

Worksheet B also relies on these 2025 references:

- Schedule 1 line 15 (SE tax)
- Schedule 2 lines 5, 6, and 13
- Form 1040 line 27a (EIC)
- Schedule 3 line 11
- Schedule 3 lines 5a, 6c, 6g, and 6h

Auto-computation deferred; manual CLW-B override field exists.

## 6. Part II-A — Additional Child Tax Credit for All Filers

ACTC exists only for qualifying children. ODC never becomes refundable.

### 6a. Preconditions

Set `ACTC = 0` if any of the following apply:

- `line4 = 0` (no CTC-eligible children);
- the return is filing **Form 2555** (IRC §24(d)(5)(B) — Form 2555 filer is barred from ACTC; spouse Form 2555 MFS-shadowed via 19 #1);
- the return fails the 2025 taxpayer SSN rule for CTC/ACTC (G7);
- `line12 <= line14` (CTC fully absorbed; no excess);
- `electsNoActc = true` (G4 design decision; no IRS basis but kept as conservative opt-out).

`Schedule 8812 line 15` is **reserved for future use** in 2025 — **not** an opt-out checkbox.

### 6b. Lines 16 through 20

```text
line16a = max(0, line12 - line14)
line16b = line4 * 1700                                        (★ 2025 OBBBA ACTC max)
line17  = min(line16a, line16b)
```

`line18a` is earned income for ACTC purposes (= `income.totalWages + income.nontaxableCombatPayElection`). It is not always identical to Form 1040 line 1z; the official earned-income chart or worksheet controls when optional self-employment methods or treaty exclusions apply. The application currently stores nontaxable combat pay inline (not separately in line 18b).

`line18b` is the total **nontaxable combat pay** received in 2025 (currently merged into line 18a).

The 2025 form then uses **line 18a**, not `line18a + line18b`, for line 19:

```text
if line18a <= 2500:
    line19 = 0
    line20 = 0
else:
    line19 = line18a - 2500
    line20 = line19 * 0.15
```

## 7. Part II-B — Three or More Qualifying Children or Puerto Rico Routing

The 2025 routing is:

```text
if line16b < 5100:                                            (= 3 × $1,700)
    if bonaFideResidentOfPuertoRico:                          (G11 fix — PR bypasses 3-child gate)
        go_to_line21
    else:
        line27 = min(line17, line20)                          (fewer than 3 children case)
else:
    if line20 >= line17:
        line27 = line17                                       (Part II-A sufficient)
    else:
        go_to_line21                                          (Part II-B for 3+ children)
```

For filers who continue into Part II-B:

```text
line21 = withheld_social_security_medicare_and_additional_medicare_tax     (W-2 box 4 sum)
line22 = schedule1.line15 + schedule2.line5 + schedule2.line6 + schedule2.line13
line23 = line21 + line22
line24 = form1040.line27a + schedule3.line11                  (EIC + AOTC; G1 fix pre-set)
line25 = max(0, line23 - line24)
line26 = max(line20, line25)                                  (alternative ACTC base)
line27 = min(line17, line26)                                  (ceiling cap)
Form1040.line28 = line27
```

## 9. Validation Rules

```text
line14 >= 0
line14 <= line12
line14 <= line13
line27 >= 0
line27 <= line4 * 1700
line27 = 0 if line4 = 0
line27 = 0 if Form2555_is_filed_by_taxpayer                   (★ form2555Spouse MFS-shadowed by 19 #1)
line27 = 0 if claimant fails 2025 CTC/ACTC SSN rule           (G7 ITIN; G9 MFJ both-ITIN rule)
line27 = 0 if electsNoActc = true                             (G4 design decision)
```

Additional validation:

- A dependent cannot be checked for both `Child tax credit` and `Credit for other dependents` (mutual exclusion structurally enforced in counting loop).
- If a child lacks a valid SSN but has a timely SSN/ITIN/ATIN, the child may still qualify for ODC even though not for CTC/ACTC.
- If the return claims CTC or ACTC, enforce the new 2025 filer-identification rule before computing the credit.
- Form 8862 recertification gate emits `FORM_8862_CTC_REQUIRED` blocking flag when prior denial flagged without Form 8862.
