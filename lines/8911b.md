# Form 8911 — Alternative Fuel Vehicle Refueling Property Credit (2025) Research Notes

Sources: 2025 Form 8911 instructions; 2025 Form 8911 (Rev. December 2024); Schedule A (Form 8911) draft (Rev. December 2025); 2025 Schedule 3 (Form 1040); Taxes For Dummies 2024 Edition.

## What Form 8911 is for

Form **8911** is used to compute the **Alternative Fuel Vehicle Refueling Property Credit** for property placed in service during the tax year. The IRS says:

* the **business/investment-use** part is treated as a **general business credit**, and
* the **personal-use** part is treated as a **personal credit**.

Taxes For Dummies matches that high-level picture and also notes the consumer summary that personal-use property was 30% up to $1,000 and business/depreciable property had a higher cap, with no carryforward for the unused personal portion.

## How many Forms 8911 per return

Usually:

```text
Form8911 count per return = 0 or 1
ScheduleA_Form8911 count per return = number of qualifying refueling properties
```

The 2025 form itself says to enter the total number of qualifying refueling properties and to **complete and file a separate Schedule A (Form 8911) for each qualifying property**. The Schedule A says the same thing: one separate Schedule A for **each** qualified refueling property placed in service during the year.

## Is it required per spouse?

No — **not per spouse** on a joint return.

On **MFJ**, you generally file **one Form 8911** for the joint return, with one Schedule A per property. The form is return-level, not spouse-level.

If spouses file **separate returns**, each spouse may file their own Form 8911 only for the property credit claimed on that spouse's own return.

## Does it require information from both spouses?

On an **MFJ** return, **yes, it can**, but still on **one joint Form 8911**.

Why:

* the personal-use limitation uses **Form 1040 line 16 + Schedule 2 line 1z** and then subtracts other return-level credits on **Form 1040 line 19** and **Schedule 3 lines 2–5 and 7** (with some exclusions), so on MFJ those are **joint** amounts.
* if either spouse owns or uses the property, or the property is jointly owned, that property still gets handled on a single joint Form 8911 and its own Schedule A.

A safe implementation rule is:

```text
if filing_status == MFJ:
    create one Form8911
    use joint-return tax limitation inputs
    attach one Schedule A (Form 8911) per qualifying property
else:
    use taxpayer-only return data
```

# What you need to collect

## Source statements / documents

For each refueling property, collect:

* description of the property
* property owner name/TIN if different from filer
* address or coordinates of the property
* date construction began
* date placed in service
* cost of the property
* business/investment-use percentage, if any
* section 179 deduction taken for the property, if any
* whether the project meets **prevailing wage and apprenticeship (PWA)** requirements
* **eligible census tract GEOID**
* local/state **certification or permit number**, if any
* if making an elective payment or transfer election, the **IRS-issued registration number** for that property.

## Extra forms that may be needed

Depending on facts:

* **Schedule A (Form 8911)** — one per property, always when filing Form 8911 for owned property.
* **Form 7220** — required for each property if claiming the **increased 30% business/investment-use credit** based on meeting PWA requirements.
* **Form 4562** — only if a **section 179 deduction** was taken, because Schedule A line 11 asks for the section 179 deduction from Form 4562 Part I.
* **Form 6251** or other AMT form/schedule — because Form 8911 line 8 requires **tentative minimum tax** even if AMT is not otherwise owed.
* **Form 3800** — for the business/investment-use credit path; also used if the only credit source is a K-1 pass-through credit.

## Pass-through credits

If the taxpayer's only 8911-related credit comes from:

* **Schedule K-1 (Form 1065) box 15 code AO**, or
* **Schedule K-1 (Form 1120-S) box 13 code AO**,
  the IRS says most non-partnership/non-S-corp filers do **not** need to complete Form 8911 and may report the credit directly on **Form 3800 Part III line 1s**.

So for a consumer 1040 engine, a useful rule is:

```text
if only_source_is_pass_through_K1_credit and taxpayer_is_not_partnership_or_Scorp:
    skip Form8911
    report via Form3800
```

# Key 2025 qualification rules

The 2025 instructions say qualified refueling property must generally:

* be placed in service during the tax year,
* have **original use** begin with the taxpayer,
* not be used predominantly outside the United States,
* be located in an **eligible census tract**,
* and if it is **personal-use property**, it must be installed on property used as the taxpayer's **main home**.

The instructions also say:

* the credit cannot be claimed for property placed in service **after June 30, 2026** because the law shortened the termination date, and
* for property placed in service after 2022, location in an **eligible census tract** is required.

# Credit percentages and caps for 2025

## Business/investment-use property

* **6%** of cost, or
* **30%** if the PWA requirements are met,
* capped at **$100,000 per single item of qualified property**.

## Personal-use property

* **30%** of cost,
* capped at **$1,000 per single item of qualified property**,
* and only if placed in service at the taxpayer's **main home**.

Also:

* property cost must be reduced by any **section 179 deduction** before credit computation.
* unless the taxpayer elects not to claim the credit, basis must be reduced by the amounts on **Schedule A line 16 and line 21** for that property.
* if the property later stops qualifying, **recapture** may apply.

# How to complete Schedule A (Form 8911) for each property

## Part I — Property details

For each property, enter:

* registration number, if required
* description
* owner name/TIN if different
* location
* construction-begin date
* placed-in-service date
* eligible census tract yes/no and GEOID
* certification/permit number, if any.

### Important stop rule

If line 6a is **No** (not in an eligible census tract), stop — the property does **not** qualify.

## Part II — Business/investment-use part

For each property:

* **line 8** = cost
* **line 9** = business/investment-use percentage
* **line 10** = line 8 × line 9
* **line 11** = section 179 deduction
* **line 12** = line 10 − line 11
* **line 13** = whether PWA requirements are met
* **line 14** = line 12 × 6%, or 30% if PWA met
* **line 15** = max allowed = $100,000
* **line 16** = smaller of line 14 or line 15; carry to **Form 8911 line 1**.

### Business-use percentage rule

If the property is mixed-use, use the business percentage. If use changes during the year, compute business percentage by months-in-business over 12 months as instructed.

## Part III — Personal-use part

Only complete if there is a personal-use portion:

* **line 17** asks whether the refueling property was installed on property used as your **main home**

  * if **No**, stop; no personal-use credit
* **line 18** = line 8 − line 10
* **line 19** = line 18 × 30%
* **line 20** = max $1,000
* **line 21** = smaller of line 19 or line 20; carry to **Form 8911 line 4**.

# How to complete Form 8911

## Item A

Enter the total number of qualified refueling properties and file a separate Schedule A for each one. If the only credit is from pass-through line 2, enter **0**.

## Part I — Credit for business/investment use part

* **line 1** = total of Schedule A **line 16** across all properties
* **line 2** = pass-through credits from K-1s
* **line 3** = line 1 + line 2
  For partnerships and S corps not making a transfer election, they stop here and report on Schedule K. For individual 1040 filers, business/investment-use credit generally flows through **Form 3800**.

## Part II — Credit for personal use part

* **line 4** = total of Schedule A **line 21** across all properties
* **line 5** = regular tax before credits:

  * for 1040 filers, **Form 1040 line 16 + Schedule 2 line 1z**
* **line 6a** = foreign tax credit
* **line 6b** = total of:

  * Form 1040 line 19, and
  * Schedule 3 lines 2 through 5 and 7,
  * reduced by Schedule 3 line 6a, 6b, and 6k
* **line 7** = line 5 − lines 6a and 6b
* **line 8** = tentative minimum tax (from the applicable AMT form/schedule)
* **line 9** = line 7 − line 8; if zero or less, enter 0 and stop unless claiming line 3 business credit
* **line 10** = personal-use credit = smaller of line 4 or line 9; enter on **Schedule 3 line 6j** (or appropriate line of return).

### Important loss rule

If line 9 is smaller than line 4, the unused **personal-use** credit is **lost**. It cannot be carried back or forward.

# Schedule A line-by-line reference

## Schedule A Part I — Property Information

| Line | Description |
|---|---|
| 1 | Registration number (if IRS-issued one is required) |
| 2 | Description of the property |
| 3 | Name and TIN of owner if different from filer |
| 4 | Location (address or coordinates) |
| 5 | Date construction began |
| 6 | Date placed in service |
| 6a | Is the property located in an eligible census tract? (Yes/No — if No, stop) |
| 6b | GEOID of eligible census tract |
| 7 | Certification or permit number, if any |

## Schedule A Part II — Business/Investment Use Credit

| Line | Formula |
|---|---|
| 8 | Cost of property |
| 9 | Business/investment-use percentage |
| 10 | Line 8 × line 9 |
| 11 | Section 179 deduction from Form 4562 Part I |
| 12 | Line 10 − line 11 |
| 13 | PWA requirements met? (Yes = 30%, No = 6%) |
| 14 | Line 12 × rate (6% or 30%) |
| 15 | Maximum: $100,000 |
| 16 | min(line 14, line 15) → **Form 8911 line 1** |

## Schedule A Part III — Personal Use Credit

| Line | Formula |
|---|---|
| 17 | Installed at taxpayer's main home? (Yes/No — if No, stop) |
| 18 | Line 8 − line 10 (personal-use cost basis) |
| 19 | Line 18 × 30% |
| 20 | Maximum: $1,000 |
| 21 | min(line 19, line 20) → **Form 8911 line 4** |

# Output mapping

For an individual 1040 return:

```text
ScheduleA_Form8911.line16 (per property) → Form8911.line1 → Form8911.line3 → Form3800 [business — out of scope]
ScheduleA_Form8911.line21 (per property) → Form8911.line4 → Form8911.line10 → Schedule3.line6j → Form1040.line20
```

The personal-use portion is what matters for non-business 1040 scope, and it lands on **Schedule 3 line 6j**.

# Bottom-line answers

## How many Form 8911 with a tax return?

* Usually **0 or 1 Form 8911 per return**
* plus **one Schedule A (Form 8911) per qualifying property**.

## Is it required per spouse?

* **No**, not on a joint return.
* On **MFJ**, file **one** Form 8911.

## Does it require information from both spouses?

* On **MFJ**, **yes, it can**, because the limitation uses **joint-return tax and credits**.
* But it is still computed on **one joint Form 8911**, not separately per spouse.

## Main extra forms/documents to collect

* **Schedule A (Form 8911)** for each property
* **Form 7220** for each property if claiming the increased PWA-based business rate
* **Form 4562** if section 179 was taken
* **Form 6251 / AMT schedule** for line 8 tentative minimum tax
* **K-1s** if any pass-through credit exists
* property address / GEOID / permit / cost / placed-in-service date / business-use % / main-home fact.
