# Form 8919 — Uncollected Social Security and Medicare Tax on Wages

**Tax year:** 2025
**IRS form:** Form 8919 (2025), Attachment Sequence No. 72, OMB No. 1545-0074
**Sources:** `docs/IRS-Forms/f8919.pdf`; 2025 Form 1040 Instructions (line 1g, Schedule 2 line 6); J.K. Lasser's Your Income Tax 2025 Professional Edition (uncollected Social Security/Medicare); IRS Schedule 2 (Form 1040) 2025
**Destination (line 1g):** Form 8919 line 6 → Form 1040 line 1g
**Destination (Schedule 2):** Form 8919 line 13 → Schedule 2 (Form 1040) line 6
**Destination (Form 8959):** Form 8919 line 6 → Form 8959 line 3 (when Additional Medicare Tax applies)

---

## 1. Line Identity

Form 8919 computes the **employee's share of uncollected Social Security and Medicare tax on wages** for workers who performed services as employees but whose employer failed to withhold FICA taxes.

The form covers:
- **Line 6** — Total wages subject to uncollected SS and Medicare tax (→ Form 1040 line 1g)
- **Lines 7–13** — Computation of uncollected SS tax (lines 7–11) and Medicare tax (line 12), and total (line 13 → Schedule 2 line 6)

Form 8919 is filed **once per spouse who qualifies** on an MFJ return (per-spouse multiplicity).

---

## 2. Who Must File (Trigger Rules)

File Form 8919 if **all** of the following apply to a spouse (taxpayer or, on MFJ, separately for the other spouse):

1. Performed services for a firm during the tax year.
2. The worker believes the pay was not for services as an independent contractor.
3. The firm did **not** withhold the worker's share of Social Security and Medicare taxes from the wages.
4. At least one of the reason codes (A, C, G, or H) applies (see §3).

**Do NOT use Form 8919 for:**
- Independent contractor services (use Schedule C / Schedule SE instead)
- Unreported tips (use Form 4137 instead)
- W-2 box 12 code M (uncollected SS on group-term life insurance) or code N (uncollected Medicare on group-term life) — these flow to Schedule 2 line 5

---

## 3. Reason Codes

Each firm row (lines 1–5, columns a–f) requires one of four reason codes in **column (c)**:

| Code | Meaning | IRS determination date required? | SS-8 required? |
|---|---|---|---|
| **A** | IRS issued a determination letter stating the worker is an employee | **Yes** — enter date in column (d) | No (already determined) |
| **C** | IRS issued other correspondence stating the worker is an employee | **Yes** — enter date in column (d) | No |
| **G** | Worker filed Form SS-8 but has not yet received a determination | No | **Yes** — file Form SS-8 separately, not attached |
| **H** | Worker received both a Form W-2 and a Form 1099-MISC or 1099-NEC from the same firm for the same services | No | **No** — do NOT file SS-8 for code H |

**Validation rules:**
- Code must be A, C, G, or H. Any other value is invalid → blocking flag.
- Codes A or C: `irsDeterminationDate` is **required** → blocking flag if absent.
- Code G: emit informational flag that Form SS-8 must be filed separately.
- Code H: `received1099MiscOrNec` must be true — **blocking flag if false** (rules.md §17 non-overrideable; promoted 2026-05-31). Per the 2025 Instructions for Form 8919, code H applies ONLY when the worker received both a W-2 and a 1099-MISC/1099-NEC from the same firm for the same services. Without a 1099 there is nothing to reclassify; filing as-is invites IRS document-matching correspondence (CP-2057). User must switch to code G/A/C or delete the row.

---

## 4. Per-Spouse Multiplicity

```
Form 8919 count per MFJ return = 0, 1, or 2 (one per qualifying spouse)
Form 8919 count per non-MFJ return = 0 or 1
```

On a **joint return**, each spouse who qualifies files their own Form 8919 — there is no shared or combined form. Each spouse's wages and tax are computed independently, then both line 6 totals aggregate into Form 1040 line 1g:

```
Form1040.line1g = Form8919_taxpayer.line6 + Form8919_spouse.line6
```

And both line 13 totals aggregate into Schedule 2 line 6:

```
Schedule2.line6 += Form8919_taxpayer.line13 + Form8919_spouse.line13
```

---

## 5. Per-Firm Table (Lines 1–5)

Each Form 8919 holds up to **5 firm rows** (lines 1–5). If more than 5 firms apply to one spouse, **attach additional Forms 8919** for lines 1–5 only — compute lines 6–13 on **one form only** using the combined totals from all attached forms.

### Table columns per firm row

| Column | Field | Required? | Notes |
|---|---|---|---|
| (a) | Firm name | Yes | Name and address of the firm |
| (b) | Firm's EIN (or SSN if individual) | Recommended | Enter "unknown" if unobtainable |
| (c) | Reason code | **Yes** — A, C, G, or H | See §3 |
| (d) | Date of IRS determination/correspondence | **Conditional** — required if code A or C | Leave blank for G and H |
| (e) | Check box: received Form 1099-MISC or 1099-NEC from this firm | No (checkbox) | Required context for code H |
| (f) | Total wages received from this firm with no SS/Medicare withholding and not reported on a W-2 | **Yes** | Feeds line 6 total |

## 6. Line-by-Line Computation (Lines 6–13)

### 6.1 Line 6 — Total Wages

```
line6 = Σ column(f) amounts across all firm rows (lines 1–5, all attached forms)
```

**→ Form 1040 line 1g** (and, if Form 8959 required, **→ Form 8959 line 3**)

### 6.2 Line 7 — Maximum Wages Subject to Social Security Tax

```
line7 = SS_WAGE_BASE   // 2025: $176,100
```

This is the **annual Social Security wage base** for 2025. The IRS prints this constant directly on the form. It is not user-entered.

> **2025 constant:** $176,100. The 2024 value was $168,600.

### 6.3 Line 8 — Total SS Wages/Tips/RRTA Already Subject to SS Tax

```
line8 = Σ W-2.box3 (social security wages)
      + Σ W-2.box7 (social security tips)
      + Σ RRTA compensation subject to 6.2% tier 1 tax (W-2 box 14 RRTA)
      + Form4137.line10 (unreported tips subject to Social Security)
```

All amounts are aggregated from W-2 entries attributed to this spouse (filtered by SSN). RRTA compensation is read from W-2 box 14 RRTA fields. If none of these sources are present, line 8 = 0.

### 6.4 Lines 9–13 — Tax Computation

```
line9  = max(0, line7 − line8)          // Remaining SS wage base available
line10 = min(line6, line9)              // Wages actually subject to SS tax
line11 = line10 × 0.062                 // Uncollected Social Security tax (6.2%)
line12 = line6  × 0.0145               // Uncollected Medicare tax (1.45%)
line13 = line11 + line12               // Total uncollected SS and Medicare tax
```

**→ Schedule 2 (Form 1040) line 6** (summed across both spouses on MFJ)

> **Note on line 10:** If the worker's other W-2 wages have already consumed the entire $176,100 SS wage base (line 8 ≥ line 7), then line 9 = 0 and line 10 = 0, meaning **no Social Security tax is owed** (line 11 = 0). Medicare tax (line 12) is still owed regardless, since Medicare has no wage base ceiling.

---

## 7. Form 8959 Integration (Additional Medicare Tax)

If the filer is required to file Form 8959, **Form 8919 line 6 wages are included in the Form 8959 line 3 total** (wages subject to Additional Medicare Tax from Form 8919):

```
if requires_Form8959:
    Form8959.line3 += Form8919_taxpayer.line6
    Form8959.line3 += Form8919_spouse.line6   // MFJ: add both
```

This ensures Form 8919 wages are captured in the 0.9% Additional Medicare Tax base when total Medicare wages exceed the applicable threshold ($200,000 single / $250,000 MFJ / $125,000 MFS).

## 11. Compute Order and Dependencies

| Step | Requires |
|---|---|
| Line 6 (total wages) | All firm rows entered; wages summed |
| Line 7 (SS wage base) | `ReferenceData.SOCIAL_SECURITY_WAGE_BASE` = $176,100 (2025) |
| Line 8 (SS already subject) | W-2 statements loaded + Form 4137 computed for same spouse |
| Lines 9–11 (SS tax) | Lines 6, 7, 8 complete |
| Line 12 (Medicare tax) | Line 6 complete |
| Line 13 (total) | Lines 11, 12 complete |
| Form 1040 line 1g | Both taxpayer and spouse Form 8919 line 6 summed |
| Schedule 2 line 6 | Both taxpayer and spouse Form 8919 line 13 summed |
| Form 8959 line 3 | Form 8919 line 6 available; Form 8959 required gate evaluated |

**Upstream dependencies:**
- W-2 statements (box 3, box 7, box 14 RRTA) must be loaded before line 8 can be computed.
- Form 4137 must be computed before line 8 if the filer has unreported tips.

**Downstream consumers:**
- Form 1040 line 1g → line 1z (total wages) → line 9 (total income)
- Schedule 2 line 6 → Schedule 2 total other taxes → Form 1040 line 17
- Form 8959 line 3 → Form 8959 Part I → Schedule 2 line 11 (Additional Medicare Tax)

## 12. Multi-Page Rule (> 5 Firms)

If a single spouse has more than 5 firms:

1. Complete additional Forms 8919 for the overflow firms (lines 1–5 only).
2. Compute **lines 6–13 on only one Form 8919** using the combined totals from all attached forms.
3. Line 6 on the primary form must equal the sum of all column (f) entries across all attached forms.

The backend collects all firm rows regardless of count, sums them into line 6, and computes lines 7–13 once. The PDF export does **not yet paginate** for > 5 firms (see §14 outstanding items).

---

## 13. Out of Scope

- **Independent contractor services** (misclassification where worker is actually self-employed) — Schedule C / Schedule SE path is not computed here.
- **Group-term life insurance uncollected SS/Medicare** (W-2 box 12 codes M and N) — these flow to Schedule 2 line 5, not line 6.
- **Unreported tips** — handled by Form 4137, not Form 8919.
- **Multi-page PDF generation for > 5 firms** — implemented; `buildMultiPagePdf()` chunks firms in groups of 5, fills computation lines on the first page only, merges via pdf-lib `copyPages`.
- **Form SS-8** — the form itself is out of scope; only the informational flag is emitted for code G.
