Below is a **2025-tax-year, implementation-ready** rule map for **Form 8959** (Additional Medicare Tax). Form 8959 computes the 0.9% Additional Medicare Tax (AMT) on Medicare wages, self-employment (SE) income, and Railroad Retirement Tax Act (RRTA) compensation above filing-status thresholds. It also reconciles AMT withholding. The result of the tax computation flows to **Schedule 2 line 11 → Form 1040 line 23**; the withholding reconciliation flows to **Form 1040 line 25c**.

---

# Form 1040 (2025) — Form 8959: Additional Medicare Tax

## 1) Form identity and cardinality

**Form 8959** — Additional Medicare Tax. Attached to Form 1040, 1040-SR, or 1040-NR.

**One Form 8959 per tax return** (not one per spouse). On an MFJ return, a single joint Form 8959 combines both spouses' amounts. On MFS, each spouse files their own Form 8959 on their own return if required.

```text
Form8959 count per return = 0 or 1
```

**MFJ aggregation rule**: On a joint return, include both spouses' data in every part:
- W-2 box 5 Medicare wages (Part I)
- Form 4137 line 6 unreported tips (Part I)
- Form 8919 line 6 wages (Part I)
- RRTA compensation (Part III)
- Medicare tax withheld and AMT withheld (Part V)

```text
if filing_status == MFJ:
    gather both spouses' wages, RRTA compensation, and withholding data into one Form8959
else:
    gather taxpayer-only data
```

| Part | Subject | Output |
|---|---|---|
| Part I | Medicare wages and tips | Line 7 |
| Part II | Self-employment income | **Out of scope** (no Schedule SE) |
| Part III | RRTA compensation | Line 17 |
| Part IV | Total AMT | Line 18 = line 7 + line 13 + line 17 → Schedule 2 line 11 |
| Part V | Withholding reconciliation | Line 24 → Form 1040 line 25c |

---

## 2) Formula

```text
Form8959.line18 (total AMT) =
    Part_I_tax      (Medicare wages/tips)
  + Part_II_tax     (SE income — OUT OF SCOPE, = 0)
  + Part_III_tax    (RRTA compensation)

Schedule2.line11    = Form8959.line18
Form1040.line23    += Schedule2.line11      (additive with other Schedule 2 other-tax items)
Form1040.line25c   += Form8959.line24       (withholding reconciliation — Part V)
```

## 3) Thresholds (2025 — NOT inflation-indexed)

| Filing Status | Threshold |
|---|---|
| Married Filing Jointly (MFJ) | $250,000 |
| Married Filing Separately (MFS) | $125,000 |
| Single | $200,000 |
| Head of Household (HOH) | $200,000 |
| Qualifying Surviving Spouse (QSS) | $200,000 |

**Critical independence rule**: The threshold is applied **separately** for wages/SE (combined, Part I + II) and for RRTA (Part III). Wages do NOT reduce the RRTA threshold; RRTA does NOT reduce the wages/SE threshold.

**SE threshold reduction** (Part II, out of scope): When SE is in scope, the SE threshold is reduced by Part I line 4 (Medicare wages). Since SE is out of scope here, this cross-part reduction is irrelevant.

**Employer withholding note**: Employers begin withholding AMT only after a single employee's wages from that one employer exceed $200,000. This is why some MFJ filers owe AMT (combined wages exceed $250k) even though no AMT was withheld.

---

## 4) Filing triggers (must-file conditions)

Form 8959 must be filed if any of the following apply:
- Medicare wages and tips on any single W-2 box 5 exceed $200,000
- RRTA compensation on any single W-2 box 14 exceeds $200,000
- Total Medicare wages + tips + SE income (both spouses on MFJ) exceed the filing-status threshold
- Total RRTA compensation (both spouses on MFJ) exceeds the filing-status threshold
- Any AMT withholding exists (Part V line 24 > 0)

## 6) Part I — Additional Medicare Tax on Medicare Wages and Tips

### Input mapping

| Line | Source | Description |
|---|---|---|
| 1 | W-2 box 5 (all W-2s, both spouses if MFJ) | Medicare wages and tips |
| 2 | Form 4137 line 6 | Unreported tips subject to Medicare tax |
| 3 | Form 8919 line 6 | Wages subject to Medicare tax (misclassified workers) |
| 4 | Lines 1 + 2 + 3 | Total Medicare wages and tips |
| 5 | Filing-status threshold (table above) | AMT threshold |
| 6 | max(0, line 4 − line 5) | Excess wages above threshold |
| 7 | line 6 × 0.009 | Part I AMT |

### Computation

```text
line1 = sum(W-2 box 5, all W-2s for taxpayer SSN)
        + sum(W-2 box 5, all W-2s for spouse SSN)   [if MFJ]
line2 = Form4137.line6   (0 if no Form 4137)
line3 = Form8919.line6   (0 if no Form 8919)
line4 = line1 + line2 + line3
line5 = threshold(filing_status)
line6 = max(0, line4 − line5)
line7 = round(line6 × 0.009)
```

**Current implementation status**: Computed inline in `computeAdditionalMedicareTax()` in `TaxReturnComputeService.java`. Part I result is wired to `Schedule2OtherTaxes.additionalMedicareTax`. No formal `Form8959` output model is created.

## 7) Part II — Additional Medicare Tax on Self-Employment Income

**OUT OF SCOPE** — No Schedule SE / Schedule C / Schedule F in this implementation. `line13 = 0` in all cases.

For reference (future implementation): wages (Part I line 4) reduce the SE threshold before applying it to SE income.

```text
line8  = max(0, ScheduleSE.line6)   // SE net income; loss → 0
line9  = filing-status threshold
line10 = line4 from Part I           // wages reduce SE threshold
line11 = max(0, line9 − line10)
line12 = max(0, line8 − line11)
line13 = round(line12 × 0.009)
```

---

## 8) Part III — Additional Medicare Tax on RRTA Compensation

RRTA compensation appears in W-2 box 14 with codes such as **RRTA Compensation**, **RRT**, or **TIER1** depending on the railroad employer. The AMT threshold for RRTA is applied **completely independently** from Part I — wages do NOT reduce the RRTA threshold.

### Input mapping

| Line | Source | Description |
|---|---|---|
| 14 | W-2 box 14 RRTA codes (or Form CT-2 line 2) | Total RRTA compensation (both spouses if MFJ) |
| 15 | Filing-status threshold (same table — NOT reduced) | AMT threshold (same dollar amount as Part I line 5) |
| 16 | max(0, line 14 − line 15) | Excess RRTA above threshold |
| 17 | line 16 × 0.009 | Part III AMT |

### Computation

```text
line14 = sumW2RrtaCompensationForSsn(taxpayerSsn)
         + sumW2RrtaCompensationForSsn(spouseSsn)   [if MFJ]
line15 = threshold(filing_status)   // same threshold as Part I — NOT reduced by wages
line16 = max(0, line14 − line15)
line17 = round(line16 × 0.009)
```

**Data availability**: `sumW2RrtaCompensationForSsn()` already exists in `TaxReturnComputeService.java` (~line 13243). It reads W-2 box 14 entries with RRTA-coded labels.

## 9) Part IV — Total Additional Medicare Tax

```text
line18 = line7 + line13 + line17
       = line7 + 0 + line17         (SE out of scope)

→ Schedule2.otherTaxes.additionalMedicareTax    = line18
→ Schedule2.otherTaxes.totalOtherTaxes         += line18
→ Form1040.taxAndCredits.otherTaxes            += line18   (line 23)
```

## 10) Part V — Withholding Reconciliation

Part V isolates the **AMT amount withheld** from the combined Medicare withholding reported in W-2 box 6 (which includes both the regular 1.45% employee share and any AMT withholding over $200k). The result flows to **Form 1040 line 25c**, not line 23.

### Input mapping

| Line | Source | Description |
|---|---|---|
| 19 | W-2 box 6 total (+ box 12 code B/N if applicable) | Total Medicare tax withheld (regular + AMT combined) |
| 20 | W-2 box 5 total (= Part I line 1) | Medicare wages (used to compute expected regular withholding) |
| 21 | line 20 × 0.0145 | Expected regular Medicare withholding (1.45% employee share) |
| 22 | max(0, line 19 − line 21) | AMT withheld on wages (box 6 excess over regular) |
| 23 | W-2 box 14 RRTA AMT withheld (or Form CT-2 line 3) | RRTA AMT withheld |
| 24 | line 22 + line 23 | Total AMT withheld → Form 1040 line 25c |

### Computation

```text
line19 = sum(W-2 box 6, all W-2s, both spouses if MFJ)
line20 = line1   // same as Part I line 1
line21 = round(line20 × 0.0145)
line22 = max(0, line19 − line21)
line23 = sum(W-2 box 14 RRTA AMT withholding, both spouses if MFJ)
line24 = line22 + line23

→ Form1040.payments.additionalMedicareTaxWithheld = line24
→ Form1040.line25c += line24
```

## 11) Compute order and dependencies

| Step | Action |
|---|---|
| 1 | Form 4137 must be computed first (Part I line 2 dependency) |
| 2 | Form 8919 must be computed first (Part I line 3 dependency) |
| 3 | Compute Part I lines 1–7 (W-2 box 5 + Form 4137 + Form 8919) |
| 4 | Part II = 0 (SE out of scope) |
| 5 | Compute Part III lines 14–17 via `sumW2RrtaCompensationForSsn()` |
| 6 | Compute Part IV line 18 = line 7 + line 17 |
| 7 | Wire line 18 → Schedule 2 line 11 → Form 1040 line 23 (additive) |
| 8 | Compute Part V lines 19–24 (withholding reconciliation) |
| 9 | Wire line 24 → Form 1040 line 25c (additive) |

**Upstream dependencies**: Form 4137, Form 8919, W-2 statement data (boxes 5, 6, 14).

**Downstream consumers**: Form 1040 line 23 (via Schedule 2 line 11), Form 1040 line 25c.

## 13) Validations

| Rule | Description |
|---|---|
| Line 6 floor | `max(0, ...)` — wages below threshold produce 0, not negative |
| Line 16 floor | `max(0, ...)` — RRTA below threshold produces 0, not negative |
| Line 22 floor | `max(0, ...)` — AMT withheld cannot exceed total box 6 |
| Threshold independence | Part I and Part III thresholds are the same dollar amount applied independently; wages do NOT reduce RRTA threshold |
| MFJ aggregation | Both spouses' wages, RRTA, and withholding combined into one form |
| MFS split | Each spouse files their own return with $125k threshold; no cross-spouse aggregation |
| Form 4137 ordering | Form 4137 must be computed before Form 8959 |
| Form 8919 ordering | Form 8919 must be computed before Form 8959 |
