# Schedule 8812 (Form 1040) — 2025
## Credits for Qualifying Children and Other Dependents

> Source: 2025 Schedule 8812 (Form 1040), Created 7/30/25 (Cat. No. 59761M).
> Authoritative sources: [2025 Schedule 8812](https://www.irs.gov/pub/irs-pdf/f1040s8.pdf) · [2025 Instructions](https://www.irs.gov/pub/irs-pdf/i1040s8.pdf)
> This form is a **computed output** of the tax return — no user fills it in directly.
> It produces two Form 1040 outputs: **line 19** (nonrefundable CTC+ODC) and **line 28** (refundable ACTC).

## 2. Overview — What Schedule 8812 Computes

```
Part I    (lines 1–14)  → Schedule 8812 line 14 → Form 1040 line 19  (nonrefundable CTC+ODC)
Part II-A (lines 15–20) \
Part II-B (lines 21–26)  → Schedule 8812 line 27 → Form 1040 line 28 (refundable ACTC)
Part II-C (line 27)     /
```

**Part I** computes the nonrefundable credit by:
1. Building MAGI from AGI + Form 2555 exclusions
2. Computing tentative credit = (CTC children × $2,200) + (ODC dependents × $500)
3. Reducing for phase-out
4. Capping at available tax liability (Credit Limit Worksheet A)

**Part II** computes the refundable ACTC only if Part I leaves excess credit beyond the tax liability cap. It is earned-income-limited and blocked entirely for Form 2555 filers.

## 4. Qualifying Dependent Rules

### 4.1 Child Tax Credit (CTC) — Line 4

A qualifying child for CTC must:
- be your **dependent**,
- be **under age 17 at the end of 2025**,
- meet the dependent conditions under the Form 1040 instructions,
- have a **valid SSN** issued by the return due date (ITIN does not qualify for CTC).

**2025 MFJ rule**: For joint filers, only one spouse must have a valid SSN; the other spouse must have an SSN or ITIN issued by the due date. Each CTC child still requires a valid SSN.

**Born and died in 2025**: If a qualifying child was born and died in 2025 and has no SSN, attach a birth certificate, death certificate, or hospital records showing the child was born alive.

### 4.2 Credit for Other Dependents (ODC) — Line 6

A qualifying other dependent for ODC must:
- be your **dependent**,
- **not** qualify for CTC/ACTC (either age 17+ or lacks valid SSN),
- be a **U.S. citizen, U.S. national, or U.S. resident alien**,
- have an **SSN, ITIN, or ATIN** issued by the return due date.

A dependent cannot have both the "Child tax credit" and "Credit for other dependents" boxes checked on Form 1040.

## 5. Part I — Child Tax Credit and Credit for Other Dependents (Lines 1–14)

### 5.1 MAGI Computation

```
line1  = form1040.line11b                      // AGI — use codebase field line11b
                                               // (IRS form label: line 11)

line2a = 0                                     // Puerto Rico excluded income — out of scope
line2b = (form2555Taxpayer.line45 ?? 0)        // Foreign earned income exclusion
       + (form2555Taxpayer.line50 ?? 0)        // Housing exclusion
       + (form2555Spouse.line45   ?? 0)
       + (form2555Spouse.line50   ?? 0)
line2c = 0                                     // Form 4563 line 15 — out of scope
line2d = line2a + line2b + line2c

line3  = line1 + line2d                        // Modified AGI (MAGI) for CTC purposes
```

### 5.2 Qualifying Counts

```
line4  = numQualifyingChildren
         // Count of dependents where qualifiesChildTaxCredit = true
         // Each must have "Child tax credit" box checked on Form 1040 Dependents section
         // Must have valid SSN; ITIN does not qualify for CTC

line6  = numOtherDependents
         // Count of dependents where qualifiesCreditOtherDependents = true
         // Each must have "Credit for other dependents" box checked
         // Cannot check both CTC and ODC box for the same person
         // SSN, ITIN, or ATIN acceptable for ODC
```

### 5.3 Tentative Credit

```
line5  = line4 × 2200                          // CTC component (2025: $2,200/child)
line7  = line6 × 500                           // ODC component ($500/dependent)
line8  = line5 + line7                         // Total potential credit before phase-out
```

### 5.4 Phase-Out

```
line9  = 400000  if filingStatus == MFJ
         200000  otherwise  (Single, MFS, HOH, QSS)

// Line 10: excess MAGI, rounded UP to next $1,000
excess = line3 - line9
if excess <= 0:
    line10 = 0
else:
    line10 = ceiling(excess / 1000) × 1000
    // Examples: excess $425 → $1,000; excess $1,025 → $2,000

line11 = line10 × 0.05                         // = $50 per $1,000 of excess MAGI
```

### 5.5 Credit After Phase-Out — Gate Check

```
// Line 12 gate:
if line8 <= line11:
    line12 = 0
    // STOP — no CTC, no ODC, no ACTC
    // line14 = 0, line27 = 0
    // Form1040.line19 = 0, Form1040.line28 = 0
else:
    line12 = line8 - line11
```

### 5.6 Credit Limit Worksheet A (→ Line 13)

This worksheet is in the Schedule 8812 instructions (not on the form itself). It computes available tax after other nonrefundable credits that precede CTC in the ordering:

```
wA_1 = form1040.line18                         // Total tax before credits (line 16 + line 17)
wA_2 = schedule3.foreignTaxCredit        ?? 0  // Schedule 3 line 1 (Form 1116 — deferred)
wA_3 = schedule3.childDependentCareCredit ?? 0 // Schedule 3 line 2 (Form 2441 — computed)
wA_4 = schedule3.educationCredits        ?? 0  // Schedule 3 line 3 (Form 8863 — deferred)
wA_5 = schedule3.retirementSavingsCredit ?? 0  // Schedule 3 line 4 (Form 8880 — deferred)
wA_6 = schedule3.elderlyDisabledCredit   ?? 0  // Schedule 3 line 6d (Schedule R — deferred)
wA_7 = wA_2 + wA_3 + wA_4 + wA_5 + wA_6
wA_8 = max(0, wA_1 - wA_7)

line13 = wA_8                                  // Credit limit (= line 13 on form)
```

> **Current state**: `schedule3.childDependentCareCredit` is the only Worksheet A input currently computed (wired from Form 2441). All others default to null/0. This is correct for filers without those other credits.

**Credit Limit Worksheet B** is needed (instead of Worksheet A) when ALL of these apply:
- The filer claims one of: Form 8396 (mortgage interest credit), Form 8839 (adoption credit), Form 5695 Part I (residential clean energy credit), or Form 8859 (DC first-time homebuyer credit)
- The filer is **not** filing Form 2555
- Schedule 8812 line 4 is more than zero

> All Worksheet B trigger forms are **out of scope** for this implementation. Worksheet A applies to all current filers.

### 5.7 Nonrefundable CTC+ODC Credit

```
line14           = min(line12, line13)
Form1040.line19  = line14
```

If `line13 = 0` (line 18 = 0): `line14 = 0` — no nonrefundable CTC/ODC. ACTC may still be available if qualifying children exist and earned income is sufficient.

## 6. Part II-A — Additional Child Tax Credit, All Filers (Lines 15–20)

### 6.1 Preconditions — ACTC = 0 (line 27 = 0, Form 1040 line 28 = 0)

| Condition | Reason |
|---|---|
| `numQualifyingChildren = 0` (line4 = 0) | ODC-only filers cannot claim ACTC |
| `line12 = 0` | Phase-out eliminated entire credit |
| Form 2555 filer (taxpayer or spouse) | "If you file Form 2555, you cannot claim the additional child tax credit" |

If any precondition is met: skip Parts II-A and II-B entirely; `line27 = 0`.

> **2025 change**: Line 15 is "Reserved for future use." The 2024 voluntary opt-out checkbox ("I do not want to claim the additional child tax credit") has been **removed** from the 2025 form. There is no longer a user election to skip ACTC.

### 6.2 Part II-A Computation

```
line15  = (reserved — leave blank)

line16a = max(0, line12 - line14)              // Excess credit beyond tax limit
line16b = numQualifyingChildren × 1700         // ACTC ceiling by family size
          // Use same child count as line 4
          // If line16b = 0: ACTC = 0

line17  = min(line16a, line16b)                // ACTC potential from credit-excess method

line18a = earnedIncomeForActc                  // See Section 7 below
line18b = 0                                    // Nontaxable combat pay — not separately tracked

// Line 19 — earned income threshold test:
if (line18a + line18b) > 2500:
    line19 = (line18a + line18b) - 2500        // "Yes" branch
else:
    line19 = 0                                 // "No" branch — line 20 = 0

line20  = line19 × 0.15
```

### 6.3 Part II-A → Part II-C Routing (No Puerto Rico)

```
// "Next. On line 16b, is the amount $5,100 or more?"
if line16b < 5100:
    // "No" branch (fewer than 3 qualifying children):
    // Skip Part II-B; enter smaller of line 17 or line 20 on line 27
    line27 = min(line17, line20)

else:
    // "Yes" branch (3+ qualifying children):
    if line20 >= line17:
        // Skip Part II-B; enter line 17 on line 27
        line27 = line17
    else:
        // Go to Part II-B (line 21)
        → compute lines 21–26; line27 = min(line17, line26)
```

> Note: The Puerto Rico bona fide resident path (which can send < 3-child filers to Part II-B) is **out of scope**.

## 7. Earned Income for ACTC (Schedule 8812 Line 18a)

Include:
- W-2 wages (Form 1040 line 1a)
- Household employee wages (line 1b)
- Tip income (line 1c)
- Medicaid waiver payments if elected (`includeQualifiedPaymentsInEarnedIncomeForEIC_ACTC = true`, line 1d)
- Dependent care benefits (line 1e)
- Adoption benefits (line 1f)
- Section 8919 wages (line 1g)
- Other earned income (line 1h)
- Elected combat pay (line 1i, if elected)

**Implementation shortcut**: Use `form1040.line1z` as line 18a. This already sums 1a–1h and includes elected combat pay. Treat line 18b = 0 unless nontaxable combat pay is separately tracked.

Exclude: All passive income, investment income, pensions, SS benefits, Form 2555 excluded amounts (moot — Form 2555 filers are blocked from ACTC entirely). Income excluded under a tax treaty is also excluded.


## 10. Form 8862 and Penalties

### Form 8862
Attach **Form 8862** if CTC/ACTC/ODC was denied or reduced after 2015 for a reason **other than** a math or clerical error, unless an exception applies. This is a user-declared condition; the intake flag is **deferred** (see Section 15).

### Refund delay
If ACTC is claimed, the IRS cannot issue the refund before **mid-February 2026**. The delay applies to the **entire refund**, not just the ACTC portion.

### Improper claims
If CTC/ACTC/ODC is improperly claimed:
- Reckless or intentional disregard of the rules → **2-year ban** on re-claiming
- Fraud → **10-year ban**
- Additional penalties may apply in both cases
