## Test 8 — `Line 13b — tips deduction phased out above $150k MAGI threshold (single)`

Command:
```powershell
npx playwright test --workers=1 --output=test-results-13b -g "tips deduction phased out above" line13b-additional-deductions.spec.ts
```

### Purpose
Verifies the **Part II floor-bucketed phaseout** at $150k threshold for a single filer: $160k MAGI is $10k over the threshold → 10 buckets × $100 = $1,000 reduction → $20,000 raw tips minus $1,000 phaseout = $19,000 final line 13.

### Hand computation per IRS rule
```
AGI                                 = $160,000
MAGI                                = $160,000

  raw tips                          = $20,000 (W-2 box 7, SSN-matched)
  cap                               = min($20,000, $25,000) = $20,000

  phaseout                          = 10 × $100 = $1,000

line38                              = $19,000
Form 1040 line 13b                  = $19,000
```
