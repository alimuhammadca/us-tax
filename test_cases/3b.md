# Test cases — Line 3b (Ordinary dividends)

**Spec files audited:**
- `e2e/tests/line3ab-dividend-income.spec.ts` (16 tests)
- `e2e/tests/line8814-child-interest-dividends.spec.ts` (15 tests)

**Total tests: 31 (all 31 passing per regression run 2026-06-03 19:xx UTC).**

**Audit method per test (IRS-first, Line 3b lens):**
1. Read inputs the test seeds.
2. Read the IRS rule that should apply for Line 3b (Form 1040 2025 instructions; Publication 17 "Dividends and Other Distributions"; Publication 550 chapter 1; Form 8814 2025 instructions; cited inline).
3. Hand-compute the expected Line 3b value.
4. Read the Java compute path (`TaxReturnComputeService.computeDividendForPerson` at `:7757-7908` and `computeDividendIncome` at `:7585-7760`) and confirm whether the code matches the IRS rule.
5. Read the test's `expect(...)` assertions that touch Line 3b directly OR indirectly (per-payer Schedule B Part II rows, line 3c checkbox, parent-isolation assertions in child-side tests).
6. Per assertion: **OK** (matches IRS for Line 3b) / **MISMATCH** (assertion contradicts IRS rule) / **GAP** (assertion missing for an output the IRS rule says Line 3b should produce).

**Scope note:** Line 3b shares its compute path with Line 3a (both are produced by `computeDividendForPerson` in the same pass). The audit at `test_cases/3a.md` already exercises every test from the Line-3a angle. This document records the Line-3b angle, which is **mostly the same compute path** but with these distinct invariants:
- **Subset rule** — Line 3a ≤ Line 3b (qualified is a subset of ordinary).
- **Sum-to-aggregate** — Line 3b is the unsigned aggregate (per-person ordinary sum after nominee subtraction, then summed across both spouses under Married Filing Jointly, plus Form 8814 line 9 symmetric contribution).
- **Schedule B trigger** — Line 3b > $1,500 OR any nominee signal present.
- **Line 9 (total income) consumption** — Line 3b feeds Line 9; Line 3a does NOT (would double-count).

**Key IRS constants used in this audit (TY 2025):**
- Schedule B Part II trigger threshold: **$1,500** (2025 Schedule B instructions).
- Form 8814 line 5 base amount (income exclusion floor): **$2,700**.
- Form 8814 child-income gross-income ceiling: **$13,500**.
- Whole-dollar rounding: round-half-to-even applied once at the end.
