## Glossary

- **Box 1 — Rollover** disclosure annotation. Fires per Internal Revenue Code §408(d)(3).
- **Box 2 — QCD** (Qualified Charitable Distribution) disclosure annotation. Fires per Internal Revenue Code §408(d)(8).
- **Box 3 — Other write-in** disclosure annotation. Fires for Health Savings Account Funding Distribution (Internal Revenue Code §408(d)(9), literal text "HFD") and/or user-entered other write-in.
- **Exception 2** — Form 8606 basis tracking (Internal Revenue Code §72). NOT a Line 4c box — Form 8606 itself serves as the IRS disclosure.
- **Breakout statement** — IRS-required attachment when more than one Line 4 exception applies; suppressed by the 2025 "Exception 2 + one other" waiver.
- **Married Filing Separately (MFS)** — filing status that suppresses spouse-side contributions to all Line 4c outputs via the single-guard cascade.

## Reference data (TY 2025)

Line 4c itself has no monetary constants. The Line-4c-adjacent flags reference:

- Qualified Charitable Distribution annual cap per person — **$108,000** (Internal Revenue Code §408(d)(8)(A); SECURE 2.0 §307). Stored at `ReferenceData.QCD_ANNUAL_CAP_PER_PERSON_2025`.
- Qualified Charitable Distribution split-interest-entity lifetime cap per person — **$54,000** (Internal Revenue Code §408(d)(8)(F)). Stored at `ReferenceData.QCD_SIE_LIFETIME_CAP_PER_PERSON_2025`.

**Line 4c coverage:** N/A (compute not reached).

**Gap analysis:** None — pre-compute gate test, intentionally does not exercise Line 4c outputs.

**Status:** Clean.
