- **OK** — assertion value matches what the IRS rule would compute by hand, AND the code computes that same value

- **GAP** — the test is silent on an output the IRS rule says should be populated, or the assertion is too lenient (e.g., `toBeFalsy()` when IRS would specify a precise value)

### Purpose

### Inputs

| Field | Value |
|---|---|

### IRS rule (what we expect for Line 6b)

Per Pub. 915 §2.4 and the IRC §86(e) "Lump-Sum Election Method" Worksheet 3:

1. **Current-year regular benefits** = $30,000 − $5,000 lump-sum = $25,000. Run the standard worksheet on $25,000 with current-year income context.
2. **Prior-year addition** = recompute 2024's would-have-been-taxable benefit if $5,000 had been received that year, subtract the $3,000 actually-reported-taxable-then. Take the positive difference.
3. **Election method result** = (1) + (2).
4. **Normal method result** = run the standard worksheet on the full $30,000.
5. **Line 6b** = min(normal, election). Line 6c is checked iff election < normal.

For MFS-lived-apart-all-year: filing-status group is **Single-equivalent** (base $25,000 / second-tier $34,000), so $25k base applies (NOT the MFS-with-spouse $0 base). Line 6d = true.

### By-hand expected outputs

Worksheet line 3 = wages 0 + interest 0 + dividends 0 + IRA 0 + pension 0 + cap-gain 0 + line 8 = $30,000 (from `manualOtherIncomeLine8`). Worksheet line 5 = $0 (no Form 2555/8839/4563/possession).

**Standard worksheet on full $30,000:**
- w7 (provisional income) = $30,000 + $15,000 (50% SS) = $45,000
- overBase = $45,000 − $25,000 = $20,000 > 0
- overSecond = $45,000 − $34,000 = $11,000 > 0 → plateau branch
- plateau = min(0.5 × ($34k − $25k), 0.5 × $30k) = min($4,500, $15,000) = $4,500
- taxable (pre-cap) = $4,500 + 0.85 × $11,000 = $14,150
- 85% cap = 0.85 × $30,000 = $25,500
- normal = min($14,150, $25,500) = **$14,150**

**Lump-sum election on $25,000 current-year + $5,000 prior:**
- Current-year portion: w1 = $25,000, w2 = $12,500. Provisional = $30,000 + $12,500 = $42,500. overBase = $17,500. overSecond = $8,500. plateau = min($4,500, $12,500) = $4,500. taxable = $4,500 + 0.85 × $8,500 = $11,725. Cap = 0.85 × $25,000 = $21,250. Current-year portion = $11,725.
- 2024 prior-year recompute: $5,000 SS allocated to 2024, no prior-year other income, no prior-year exclusions, current-year filing status (MFS-lived-apart Single-equivalent). Provisional = 0 + 0.5 × $5,000 = $2,500. Below $25k base → taxable from $5k = $0. Subtract $3,000 already reported as taxable in 2024 → max(0, 0 − 3000) = $0.
- Election total = $11,725 + $0 = **$11,725**.

**Line 6b** = min($14,150, $11,725) = **$11,725**. Line 6c = true (election won). Line 6d = true (MFS-lived-apart).

**Form 1040 line 9 (total income)** = line1z 0 + line2b 0 + line3b 0 + line4b 0 + line5b 0 + line6b 11,725 + line7a 0 + line8 30,000 = **$41,725**.

### Code path
