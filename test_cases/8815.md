## Test 4 — MFS suppression + `FORM_8815_MFS_BLOCKED` flag emission

**Command:**
```powershell
npx playwright test --workers=1 --output=test-results-8815-t4 -g "MFS filer claim is suppressed" line2ab-form8815-fullflow.spec.ts
```

**Purpose.** Lock in IRC §135(d)(2): MFS filers cannot take the savings bond exclusion. The application must (a) return `form8815 = null` and (b) emit a `FORM_8815_MFS_BLOCKED` informational flag so the user knows their claim was suppressed.

**Inputs seeded.** Filing status `Married filing separately`, plus `claimsSavingsBondExclusionForm8815=true` and `savingsBondExclusionAmount=500` (any non-null amount should still be suppressed).

**IRS rule:** IRC §135(d)(2): "Subsection (a) shall not apply to any taxpayer for any taxable year if such taxpayer is a married individual filing a separate return for the taxable year." → Exclusion forbidden.

**Per-assertion verdict.**

| Assertion | IRS-rule value | Test expects | Verdict |
|---|---|---|---|
| `form8815 ?? null` is null | null | null | **OK** |
| `FORM_8815_MFS_BLOCKED` flag present in `flags` | present | truthy | **OK** |
